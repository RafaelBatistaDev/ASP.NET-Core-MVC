## [1.0.0] - 2026-04-04

### ✨ Added
- ✅ CRUD completo para gerenciamento de filmes (Create, Read, Update, Delete)
- ✅ Entity Framework Core com SQLite integrado
- ✅ Migrações de banco de dados automáticas
- ✅ Validação de dados robusta (Data Annotations)
- ✅ Pesquisa e filtragem avançadas
- ✅ Interface responsiva com Bootstrap 5
- ✅ Padrão MVC profissional
- ✅ CSRF Token em formulários (segurança)
- ✅ Layout master com CSS customizado
- ✅ Static assets otimizados

### 🔧 Changed
- Framework atualizado para ASP.NET Core (versão mais recente)
- Utilização de C# moderno com nullable reference types
- Appsettings organizado para dev/prod
- Estrutura de projeto otimizada
